# Ava
A Discord bot written in Go. Named after the AI Ava from *Ex Machina*

Add the environment variable `BOT_TOKEN` in `.env` as your bot's secret token. Copy the `docker-compose.yml` file to your directory and use 
```
# docker compose up
```
to launch the bot.
